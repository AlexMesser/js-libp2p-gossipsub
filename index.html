Intro

Installable via npm install --save libp2p-gossipsub, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save libp2p-gossipsub

You can then require() libp2p-gossipsub as normal:

const libp2PGossipsub = require('libp2p-gossipsub')

In the Browser

libp2p-gossipsub should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of libp2p-gossipsub, including index.js and index.min.js, are included in the /dist folder. libp2p-gossipsub can also be found on unpkg.com under

  • https://unpkg.com/libp2p-gossipsub/dist/index.min.js
  • https://unpkg.com/libp2p-gossipsub/dist/index.js
